Fully GCP-native observability — no third-party APM. Istio sidecars emit traces and metrics automatically. All application logging is structured JSON (slog in Go, structlog in Python). Logs sink to BigQuery for compliance retention. Uptime checks probe all public endpoints from multiple regions. Per-environment budget alerts track spend. See Architecture Notes for alert policies and retention details.
