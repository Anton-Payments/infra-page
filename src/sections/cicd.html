GitHub Actions never touches the cluster. CI builds and pushes images, then updates the image tag in the deploy repo. ArgoCD is the only system with cluster credentials. Each service repo (api, anton-engine, merchant, operations, www) has its own CI pipeline. Branch mapping:  main → Production + Sandbox  develop → Staging
